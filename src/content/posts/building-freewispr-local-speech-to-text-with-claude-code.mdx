I was paying ₹320/month for Wispr Flow on Mac. When I moved to Windows, there was no equivalent. So I built one.

Using Claude Code, I put together **Freewispr** in a few hours — my first desktop application. Hold `Ctrl+Space`, speak, and text appears wherever your cursor is.

## What it does

- **Local, offline transcription** — no cloud dependency, no internet required
- **Sub-second latency** — fast enough to feel native
- **Snippet library** — trigger words that expand into full phrases
- **Personal dictionary** — train it on custom terms and names
- **Auto-punctuation**
- **Single `.exe` deployment** — no Python, no dependencies to install

## Why local matters

Most speech-to-text tools send your audio to the cloud. Freewispr runs entirely on your machine. That means it works offline, stays private, and has no recurring cost.

## Built with Claude Code

This was my first time building a desktop app. Claude Code handled the heavy lifting — architecture, the Windows integration layer, packaging it into a single executable. A few hours from idea to working tool.
